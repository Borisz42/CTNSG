# ARMOR-MAD Orchestrator Routing

This directory contains the Adaptive Routing for Heterogeneous Multi-Agent Debate (ARMOR-MAD) logic.

## Purpose
It dynamically routes discrete graph generation tasks to the appropriate logical subsystems, such as the RelDiT topology generator, the VNPool Realizer, and the SMT Validator. It orchestrates the full end-to-end dataflow between specialized agents.
